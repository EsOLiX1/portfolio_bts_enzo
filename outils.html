Mes outils

Mes outils

Feedly

Agrégateur de flux RSS qui centralise les actualités, articles et blogs pour suivre facilement l’information en un seul endroit.

Microlink

API qui permet de générer rapidement des aperçus enrichis (cartes, métadonnées, captures) à partir d’URLs.

Firebase

Plateforme de développement Google offrant hébergement, base de données temps réel, authentification et analytics pour créer des apps web et mobiles.

GitHub

Service de gestion de code source collaboratif basé sur Git, utilisé pour héberger, versionner et partager des projets logiciels.
